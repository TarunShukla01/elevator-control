A simulation-based implementation of an Elevator Control System using React, TypeScript, functional services, and Jest for deterministic testing.
This project follows professional engineering practices with clean separation between UI, business logic, state management, and simulation timing. Designed to demonstrate real-world front-end architecture skills and mastery of TypeScript.
